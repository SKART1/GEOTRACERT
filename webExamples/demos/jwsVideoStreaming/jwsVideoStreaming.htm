jWebSocket Video & Audio Streaming Demo (Webm)

This demo shows how to stream in Real Time using jWebSocket Framework the audio and video captured from the presenter's web camera to many connected viewers. Using a system channel all the registered users into the channel will receive the video stream information, in the log box under the scenario will be shown all the incoming data and events from the channel. Don't forget to set getUserMedia enabled in Firefox by setting the media.peerconnection.enabled option to true in about:config.
